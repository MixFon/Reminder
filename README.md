# Reminder
Простое приложение для создания списка напоминаний.

Основное назначение:
1. Составление списка покупок
2. Составление списка дел
3. Составление списка вещей, которые нужно взять в поездку
4. Составление списка напоминаний

Так же приложение было написана с целью более глубоково изучения CoreData.
